GeoJSON Viewer

Interactive viewer for GeoJSON data which supports arbitrary coordinates, i.e. also cartesian coordinates. No map tiles are provided. Purpose is rather to visualize general Gemoetries, but built with geo-json format.

Image Resizer

Converts an image to JPEGs using a number of different quality settings so you can select the smallest file size that is still usefully legible. Original: Simon Willison.

Jina Reader

Convert any URL to copyable Markdown using the Jina Reader API. Original: Simon Willison.

Timestamp Converter

Convert between Unix timestamps and human-readable dates: Original: Simon Willison.

Encrypt

Encrypt / decrypt message - encrypt a message with a passphrase, send someone a link and they can decrypt it again. Original: Simon Willison.

Json to GeoJson Converter

Convert a json config to GeoJson for visualization in https://geojson.io/.
